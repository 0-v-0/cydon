Cydon使用一种全新的方法来构建Web应用程序，它不依赖于虚拟DOM，而是在绑定阶段将所有DOM可变部分收集起来，维护一个变量名到DOM可变部分的关系，并采用异步更新的方法更新DOM。

与使用虚拟（virtual）DOM 差异对比不同，在更新DOM时，由于在绑定阶段已经收集了可变部分，因而无需遍历(v)DOM树，只需按顺序更新变化的部分即可，这使得Cydon在性能和体积上都有很大的优势。
